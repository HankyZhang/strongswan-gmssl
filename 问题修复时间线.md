### 阶段 5: GmSSL HMAC API 适配 (问题 18-20)
**时间**: 第四轮编译
**问题**: GmSSL 3.x HMAC API 变更

#### 问题 18: hmac_init 参数错误
**文件**: `src/libstrongswan/plugins/gmsm/gmsm_sm3_{signer,prf}.c`

```c

```

**修复**:
```c

hmac_init(&ctx, &sm3_digest, key.ptr, key.len);
```

---

#### 问题 19: hmac_finish 参数错误
```c
error: too few arguments to function 'hmac_finish'
```

**修复**:
```c
// 旧代码

// 新代码
size_t mac_len;
hmac_finish(&ctx, mac, &mac_len);
```

---

#### 问题 20: HMAC_CTX 成员访问错误
```c
error: 'HMAC_CTX' has no member named 'key'
error: 'HMAC_CTX' has no member named 'key_size'
```

**修复**:
```c
// 添加密钥存储
struct private_gmsm_sm3_signer_t {
    HMAC_CTX hmac_ctx;
    chunk_t key;  // 新增
};

// set_key 中存储密钥
chunk_clear(&this->key);
this->key = chunk_clone(key);

// 重新初始化时使用存储的密钥
hmac_init(&ctx, &sm3_digest, this->key.ptr, this->key.len);

// destroy 中清理密钥
chunk_clear(&this->key);
```

**提交**: `de21aa069d` - "Fix GmSSL 3.x HMAC API usage"

### 模式 1: 枚举值补全
```c
switch (algorithm) {
    case EXISTING_1:

    case NEW_GM_ALGO:  // 新增国密算法
```

