### 4. src/libstrongswan/crypto/iv/iv_gen.c
**问题**: SM4 加密算法枚举未处理  
**函数**: `iv_gen_create_for_alg()`

**修复**:
```c
// SM4-CBC 使用随机IV
case ENCR_AES_CBC:
case ENCR_CAMELLIA_CBC:
case ENCR_SM4_CBC:     // 新增
    return iv_gen_rand_create();

// SM4-GCM 使用序列IV
case ENCR_AES_GCM_ICV16:
case ENCR_CHACHA20_POLY1305:
case ENCR_SM4_GCM_ICV16:   // 新增

```

**技术说明**:
- CBC模式: 随机IV防止密文攻击
- GCM模式: 序列IV（计数器），防止IV重用

## 国密算法技术总结

### SM2 椭圆曲线公钥密码 (GM/T 0003-2012)
- 曲线: 256位素数域
- 公钥: 64字节（未压缩坐标）
- 用途: 签名、密钥交换、加密

### SM3 密码哈希算法 (GM/T 0004-2012)
- 输出: 256位 (32字节)
- HMAC-SM3-96: 截取96位 (12字节)
- 用途: 数字签名、HMAC、完整性校验

### SM4 分组密码算法 (GM/T 0002-2012)
- 块大小: 128位
- 密钥长度: 128位
- 模式: CBC, GCM
- 用途: 数据加密

## 编译选项说明

strongSwan 使用严格的编译警告确保代码质量：

```
-Werror               # 所有警告视为错误
-Werror=switch        # switch必须处理所有枚举
-Wall -Wextra         # 启用所有标准警告
```

这要求所有枚举值都必须在 switch 语句中显式处理。
